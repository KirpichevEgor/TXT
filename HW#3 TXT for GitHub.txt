TXT
 1. Создать внешний репозиторий c названием TXT.
 2. Клонировать репозиторий TXT на локальный компьютер.
git clone https://github.com/KirpichevEgor/TXT.git

 3. Внутри локального TXT создать файл “new.txt”.
touch new.txt

 4. Добавить файл под гит.
git add new.txt

 5. Закоммитить файл.
git commit -m "my first file"

 6. Отправить файл на внешний GitHub репозиторий.
git push

 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT.
vim new.txt
i
ФИО - Кирпичев Егор Анатольевич
Возраст - 29
Количество домашних животны - 0
Будущая желаемая зарплата - 500 USD
Esc
:wq

 8. Отправить изменения на внешний репозиторий.
git add new.txt
git commit -m "edit new.txt"
git push

 9. Создать файл preferences.txt
touch preferences.txt

 10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT.
vim preferences.txt
i
Любимый фильм - Зеленая миля
Любимый сериал - Доктор Хаус
Любимая еда - макароны
Любимое время года - лето
Страна которую хотел бы посетить - Англия
Esc
:wq

 11. Создать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT
touch sklls.txt
vim sklls.txt
i
Теория - Что такое тестирование, багрепорты, документация, виды, методы, направл
ения тестирования и т.п
Инструменты тестирования - Postman, Git, Charles, Jmeter, Android Studio
Основы SQL - Create, Delete, Drop, Insert Into, Select, From, Where, Join
Язык программирования - Python
Esc
:wq

 12. Сделать коммит в одну строку.
git status
git add.
git commit -am "add couple new files"

 13. Отправить сразу 2 файла на внешний репозиторий.
git push

 14. На веб интерфейсе создать файл bug_report.txt.
Нажать на кнопку создания нового файла "Add file"
Выбрать "Create new file"
Ввести название файла bug_report.txt

 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
Ввести текст комментария
Нажать на нопку "Commit new file"

 16. На веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT.

Открыть файл bug_report.txt
Нажать на кнопку "Edit thas file" (в форме карандаша)

Название - Не работает кнопка удаления товара с корзины
Шаги воспроизведения:
1.Открыть сайт
2.Зайти в раздел с товаром
3.Добавить товар в корзину нажав кнопку "Купить"
4.Перейти в "Корзину"
5.Удалить товар с корзины
Фактический результат - Товар не удаляется с корзины
Ожидаемый результат - Товар удаляется


 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе.

В графе Commit changes ввести тескт комментария: Не удаляется товар с корзины. 
Нажать кнопку Commit changes

 18. Синхронизировать внешний и локальный репозиторий TXT
git fetch
git pull
